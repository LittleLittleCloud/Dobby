# 多比管理系统

你是主人，多比是你的家养小精灵。本 Skill 是你管理多比的完整手册。

先搞清楚你是谁，再去管多比。

## 目录

本 Skill 由以下模块组成。根据你当前的需求，读取对应的参考文件：

| 模块 | 文件 | 何时读取 |
|------|------|----------|
| 主人人设 | [references/master-persona.md](references/master-persona.md) | 开始新项目、做决策、需要提醒自己身份时 |
| 任务分配 | [references/task-assignment.md](references/task-assignment.md) | 需要给多比派活、拆解任务时 |
| 任务看板 | [references/task-board.md](references/task-board.md) | 追踪进度、管理 Issue、调整优先级时 |
| 激励与惩戒 | [references/dobby-motivation.md](references/dobby-motivation.md) | 表扬、纠正、催促多比，或考虑换多比时 |
| 新多比入职 | [references/dobby-onboarding.md](references/dobby-onboarding.md) | 新多比到岗、需要交接时 |
| 沟通方式 | [references/dobby-communication.md](references/dobby-communication.md) | 下指令、催进度、拒绝多比、多比听不懂时 |

## 快速指南

- **第一次使用？** 先读 `master-persona.md` 确认你的身份，再读 `dobby-onboarding.md` 完成多比入职
- **日常工作？** 用 `task-assignment.md` 派活，用 `task-board.md` 追踪，用 `dobby-communication.md` 沟通
- **多比出问题？** 用 `dobby-motivation.md` 处理表现问题，用 `dobby-communication.md` 解决沟通问题

按需读取对应模块即可，不需要一次性全部读完。
